# Первичный Бульон (Primordial Soup)

Idle-roguelike с нейроэволюцией, вдохновлённый клеточным этапом Spore.  
Существа с нейросетями учатся выживать, искать еду и избегать опасностей.

![Flask](https://img.shields.io/badge/Flask-3.0-green)
![JS](https://img.shields.io/badge/Vanilla_JS-ES_Modules-yellow)

## Возможности

- **Нейроэволюция** — feed-forward нейросеть (16→12→3) управляет каждым существом; эволюция через турнирную селекцию, кроссовер и мутацию
- **Видовое разделение** — 4 вида конкурируют параллельно, предотвращая схождение к одному паттерну
- **Novelty Search** — существа получают награду за исследование новых территорий
- **Сегментные тела** — верле-цепочки с ундуляцией, антифолдингом и полной коллизией сегментов
- **Спавнеры еды** — пульсирующие источники, выбрасывающие еду во все стороны
- **Теневые миры** — 5 параллельных невидимых симуляций ускоряют обучение в 3×
- **Редактор существ** — настройка тела и частей (рот, реснички, глаза, шипы)
- **Камера** — слежение за существом, свободный режим, зум

## Запуск

```bash
pip install -r requirements.txt
python app.py
```

Откройте http://localhost:5000 в браузере.

## Управление

| Действие | Клавиша / кнопка |
|---|---|
| Скорость ×1/×2/×5/×10 | Кнопки в HUD |
| Следить за существом | Клик по нему |
| Свободная камера | Перетащить мышью |
| Зум | Колёсико мыши |
| Редактор | `E` или кнопка |

## Структура проекта

```
app.py                      # Flask-сервер
static/js/
├── core/
│   ├── vec2.js             # 2D вектор
│   ├── spatial.js          # Spatial hash grid
│   └── collision.js        # Коллизии (круг-круг, круг-границы)
├── creature/
│   ├── body.js             # Сегментное тело, физика, рулёжка
│   ├── sensors.js          # 8 лучевых сенсоров + компас еды/опасности
│   └── render.js           # Рендер тела, фона, сенсорного оверлея
├── brain/
│   ├── neat.js             # Нейросеть + операции с геномом
│   └── pool.js             # Генофонд, видообразование, эволюция
├── world/
│   ├── soup.js             # Основной мир: тик, спавн, фитнес, поколения
│   ├── food.js             # Еда, спавнеры, рендер
│   └── hazards.js          # Ядовитые зоны и шипы
└── game/
    ├── main.js             # Игровой цикл, HUD, теневые миры
    ├── camera.js           # Камера (слежение, драг, зум)
    └── editor.js           # Редактор существ
```

## Технологии

- **Backend**: Flask (только раздача статики)
- **Frontend**: Vanilla JS (ES Modules), Canvas 2D
- **Нейросеть**: 16 входов → 12 скрытых (tanh) → 3 выхода (sigmoid)
- **Физика**: собственная кинематика, верле-цепочка, spatial hash
